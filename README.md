# iRevolution: A Data-Driven Exploration of Apple’s iPhone Impact in India

## 📌 Project Overview
This project analyzes the impact of Apple's iPhone in the Indian smartphone market using **data analytics and visualization with Tableau**. The objective is to explore sales trends, product specifications, pricing patterns, and consumer insights through interactive dashboards.

The project demonstrates how data visualization tools can help understand market behavior and technological influence.

---

## 🎯 Problem Statement
The smartphone market in India is highly competitive with several brands such as Samsung, Xiaomi, OnePlus, and Vivo. Apple iPhones are positioned in the premium segment and continue to influence the market through innovation and brand value.

The goal of this project is to analyze Apple’s iPhone presence in India using data-driven insights and visualize trends using Tableau dashboards.

---

## 🗂 Dataset Description
The dataset used in this project contains information about Apple iPhone models including:

- Product Name
- Sale Price
- MRP (Maximum Retail Price)
- Discount Percentage
- Star Rating
- Number of Ratings
- RAM Specifications

This dataset helps analyze product popularity, pricing trends, and user satisfaction.

---

## 🧹 Data Preprocessing
Before creating visualizations, the dataset was cleaned and prepared using Excel.

The preprocessing steps included:

- Removing unnecessary columns
- Handling missing values
- Standardizing data formats
- Preparing structured data for Tableau analysis

---

## 📊 Tableau Dashboards
The project includes multiple visualizations created using Tableau:

### 1️⃣ KPI Dashboard
Displays key metrics such as:
- Active Users
- Revenue Generated
- Units Sold

### 2️⃣ Model Specification Dashboard
Shows detailed information about iPhone models and their hardware specifications.

### 3️⃣ Brand Price Comparison
A visual comparison of smartphone brands based on pricing.

### 4️⃣ Battery Type Distribution
Shows the distribution of battery technologies across smartphones.

### 5️⃣ Country-wise Best Selling Smartphones
Highlights popular smartphone brands across different regions.

---

## 📖 Tableau Story
The Tableau Story combines dashboards into a narrative format to explain insights step-by-step.

It highlights Apple's market position, product trends, and consumer preferences.

---
